Introduction

Analysis of collected observations. Weaving executed code with explanatory text to produce literate programming (Knuth 1984Knuth, Donald Ervin. 1984. Literate Programming. The Computer Journal 27 (2): 97111.) is provided by the R packages knitr (Xie 2016Xie, Yihui. 2016. Knitr: A General-Purpose Package for Dynamic Report Generation in R. https://CRAN.R-project.org/package=knitr.) and the document template tufte (Xie and Allaire 2016Xie, Yihui, and JJ Allaire. 2016. Tufte: Tuftes Styles for R Markdown Documents. https://CRAN.R-project.org/package=tufte.).

This document is translated from rmarkdown (Allaire et al. 2016Allaire, JJ, Joe Cheng, Yihui Xie, Jonathan McPherson, Winston Chang, Jeff Allen, Hadley Wickham, Aron Atkins, and Rob Hyndman. 2016. Rmarkdown: Dynamic Documents for R. http://rmarkdown.rstudio.com.) source. The source can be found at LCHF Analysis.Rmd.

The %>% operator will be seen in some of the code. The package magrittr (Bache and Wickham 2014Bache, Stefan Milton, and Hadley Wickham. 2014. Magrittr: A Forward-Pipe Operator for R. https://CRAN.R-project.org/package=magrittr.) provides this operator, called a pipe, which forwards the result of a preceeding function to a suceeding function in a sinle code stream. By default, the forwarded object is piped into the succeeding functions first parameter slot. The argument can be moved to other parameter slots in the succeeding call Using the . placeholder. The pipe is used where it improves readability. A series of functions interspersed with %>% should be viewed as pipeline or processing elements, with the output of one function the input to the next.

R uses formulae to define response models. A formula is the form

response variable ~ explanatory variables

The structure of formulae is well-explained by (R TutorialsModel Formulae 2016R TutorialsModel Formulae. 2016. https://ww2.coastal.edu/kingw/statistics/R-tutorials/formulae.html. Accessed September 10.).

Heartrate at RER 0.5

For later

Heartrate at RER 0.5 is the dependent variable. The hypothesis is that the low carbohydrate/high fat diet will increase the heartrate at RER, with the body becoming using more fat than protien as the testing procedes.

Working model for analysis: (Gribble 2009Gribble, Paul. 2009. Repeated Measures ANOVA Using R. Motor Control Lab Blog. https://gribblelab.wordpress.com/2009/03/09/repeated-measures-anova-using-r/.)
